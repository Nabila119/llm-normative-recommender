# Leveraging LLMs for Agent-Based Normative Recommender Systems

## Overview
This thesis explores the use of Large Language Models (LLMs) to extract and formalize normative information from stakeholders in agent-based recommender systems. The work is developed in the context of the DJ4ME project, with case studies such as food recommendation scenarios.

## Objectives
1. Investigate methods for extracting stakeholder norms from textual datasets using LLMs.
2. Evaluate the effectiveness of LLMs in capturing ethical and behavioral guidelines.
3. Encode extracted norms into formal representations suitable for agent-based systems.
4. Integrate norms into systems such as Autonomous Jiminy and Dialogue Jiminy.
5. (Optional) Explore norm-guided explanations for agent decisions.

## Methodology
- Literature review on LLMs and normative systems
- Design of norm-mining pipelines
- Implementation using LLMs
- Empirical evaluation using DJ4ME datasets

## Expected Outcomes
- A validated methodology for norm extraction using LLMs
- Formal representations of stakeholder norms
- Improved transparency and value alignment in recommender systems
